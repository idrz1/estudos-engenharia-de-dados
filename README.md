# 🐍 Trilha de Dados - Python
Repositório dedicado aos meus estudos práticos de Python, focado na construção de uma base sólida para Engenharia de Dados.

---

## 📂 Arquivos Neste Repositório

### 🗄️ **`python_basico.py`** | *Fundamentos*
> Consolidação dos pilares da linguagem. Como já trago uma base lógica estruturada (C++), este script foca na adaptação ágil da sintaxe do Python, abordando:

* **Manipulação e fatiamento** avançado de Strings.
* **Operações em Listas e Dicionários** dinâmicos.
* **Estruturas condicionais e Loops** (`for` / `in`).

---

### 📊 **`basico_em_pandas.ipynb`** | *Extração e Transformação de Dados (ETL)*
> Simulação de um pipeline de ETL resolvendo problemas de negócio reais com a biblioteca **Pandas**:

* **Limpeza de Dados:** Padronização de strings com `.str.strip()` e `.str.title()`.
* **Cruzamento de Tabelas:** Junção de bases de clientes e vendas utilizando o conceito de chaves (`pd.merge()`).
* **Agrupamento e Análise:** Geração de insights financeiros com `.groupby()` e ordenação de resultados.
